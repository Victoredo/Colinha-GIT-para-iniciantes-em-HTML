COLINHA GIT HUB

1 - Primeiro vamos configurar o git com comandos de apontamento ao usuario em questão.

  • git config --global user.name "USUARIO DO GITHUB"
  • git config --global user.email "EMAIL DO GITHUB"

    • VAMOS INICIAR O GIT:

  • git init

    • Verificando se tudo esta ok !

  • git status

    • Agora vamos adicionar iformar ao git os arquivos que iremos subir para o repositorio remoto.

  • git add * (* significa All = Todos)
  • git commit -m "Nome do repositorio desejado"

    • Agora vamos apontar o endereço do repositorio remoto que iremos enviar os arquivos

  • git remote add origin "Linf do repositorio"

    • Agora vamos subir os arquivos

  • git push origin master

    • Como menssagem final esperamos receber ao usar o comando git status novamente :

  • git status > On branch master nothing to commit, working tree clean
